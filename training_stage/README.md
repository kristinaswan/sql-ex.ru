Задание 1. Вывести имена всех людей, которые есть в базе данных авиакомпаний. [(сайт)](https://sql-academy.org/ru/trainer/tasks/1)

<details><summary>Решение</summary>

```sql
SELECT 
  model, speed, hd 
FROM 
  pc
WHERE
  price < 500;
```
